# Problema Canibais e Missionários
Algoritmo em C que soluciona o problema dos canibais e missionários.
O problema dos canibais e missionários consiste num grupo de três canibais e três missionários, que estão de um lado do rio, e todos querem chegar até o outro lado. Porém, o barco que o grupo possui tem capacidade máxima de 2 pessoas, e caso um lado fique com mais canibais que missionários, os canibais comerão os missionários. O algoritmo busca uma maneira de transportar todos até o outro lado do rio.
